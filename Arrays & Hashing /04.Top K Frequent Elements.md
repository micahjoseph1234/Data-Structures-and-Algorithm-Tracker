# 04. Top K Frequent Elements

## Problem

Given an integer array `nums` and an integer `k`, return the `k` most frequent elements.

---

# Pattern

**HashMap + Heap (PriorityQueue)**

Whenever the question says:

```text id="3ln5yj"
Top K
Most Frequent
K Largest
```

Think:

```text id="b97f2z"
Frequency Count + Heap
```

---

# Approach 1 : Sorting Frequencies

1. Count frequencies using HashMap.
2. Convert map into a list of entries.
3. Sort by frequency.
4. Pick first `k`.

---

## Time Complexity

```text id="x4n8az"
O(n log n)
```

---

# Approach 2 : Min Heap (Optimal)

## Key Observation

We only need the **top k frequencies**, not all frequencies.

So:

* Count frequencies.
* Maintain a **Min Heap of size k**.
* Remove smaller frequencies.

---

# Algorithm

### Step 1

Count frequencies.

```java id="69o1x6"
HashMap<Integer,Integer> map
```

Example:

```text id="vgp7yj"
nums = [1,1,1,2,2,3]

1 -> 3
2 -> 2
3 -> 1
```

---

### Step 2

Create Min Heap.

```java id="e5r8q0"
PriorityQueue<Integer> pq =
        new PriorityQueue<>(
                (a,b) ->
                        map.get(a) - map.get(b)
        );
```

Heap stores numbers.

Comparison happens using frequencies.

---

### Step 3

Insert keys.

```java id="ecyxwe"
for(int key : map.keySet())
{
    pq.add(key);

    if(pq.size() > k)
    {
        pq.poll();
    }
}
```

Heap size never exceeds `k`.

---

### Step 4

Remove elements from heap.

---

# Code

```java id="s5t0n0"
public int[] topKFrequent(int[] nums, int k) {

    HashMap<Integer,Integer> map =
            new HashMap<>();

    for(int num : nums)
    {
        map.put(
                num,
                map.getOrDefault(num,0)+1
        );
    }

    PriorityQueue<Integer> pq =
            new PriorityQueue<>(
                    (a,b) ->
                            Integer.compare(
                                    map.get(a),
                                    map.get(b)
                            )
            );

    for(int key : map.keySet())
    {
        pq.add(key);

        if(pq.size() > k)
        {
            pq.poll();
        }
    }

    int[] ans = new int[k];

    for(int i=0;i<k;i++)
    {
        ans[i] = pq.poll();
    }

    return ans;
}
```

# Follow-Up : Bucket Sort (Know the Concept)

Frequency can never exceed:

```text id="8r0m4v"
n
```

Create:

```java id="65g93g"
List<Integer>[] bucket =
        new ArrayList[n+1];
```

Index:

```text id="ljlwmh"
bucket[frequency]
```

Example:

```text id="l3s9qk"
1 -> 3
2 -> 2
3 -> 1
```

Store:

```text id="d3o0qg"
bucket[3] = [1]
bucket[2] = [2]
bucket[1] = [3]
```

Traverse from end.
