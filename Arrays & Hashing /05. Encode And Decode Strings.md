# 05. Encode And Decode Strings

## Problem

Design an algorithm to encode a list of strings into a single string and decode it back to the original list.

The encoded string should be able to represent:

```text
["hello", "world"]
["ab#cd", "xyz"]
["", "abc"]
```

without losing information.

---

# Pattern

**String Encoding + Parsing + Two Pointers**

Whenever the question says:

```text
Convert multiple strings into one string
and recover them later
```

Think:

```text
Metadata + Parsing
```

---

# Initial Idea (Wrong)

Use:

```text
ab#xyz#
```

and split whenever `#` is found.

---

## Why does it fail?

Input:

```text
["ab#cd", "xyz"]
```

Encoding:

```text
ab#cd#xyz#
```

During decoding:

```text
ab
cd
xyz
```

Wrong.

The original strings were:

```text
["ab#cd", "xyz"]
```

---

# Key Observation

Constraint:

```text
Strings can contain any ASCII character.
```

Therefore:

```text
No delimiter is safe.
```

Not:

```text
#
##
$
@@
```

Nothing is safe.

---

# Correct Idea

Store:

```text
length#string
```

Example:

```text
["neet","code"]
```

becomes:

```text
4#neet4#code
```

---

# Why does this work?

Suppose:

```text
5#ab#cd3#xyz
```

Read:

```text
5
```

Take next:

```text
5 characters
```

Result:

```text
ab#cd
```

Even though `#` exists inside the string, there is no ambiguity.

---

# Encoding Algorithm

For every string:

1. Find its length.
2. Append length.
3. Append `#`.
4. Append the string itself.

---

# Encode Code

```java
public String encode(List<String> strs) {

    StringBuilder sb =
            new StringBuilder();

    for(String s : strs)
    {
        sb.append(s.length());
        sb.append("#");
        sb.append(s);
    }

    return sb.toString();
}
```

---

# Encoding Example

Input:

```text
["ab","xyz"]
```

Output:

```text
2#ab3#xyz
```

---

# Decoding Algorithm

Use two pointers.

Example:

```text
2#ab3#xyz
```

### Step 1

Read until `#`.

```text
2
```

Length:

```text
2
```

Take next:

```text
2 characters
```

Result:

```text
ab
```

---

### Step 2

Move pointer.

Read:

```text
3
```

Take next:

```text
3 characters
```

Result:

```text
xyz
```

---

# Decode Code

```java
public List<String> decode(String str) {

    List<String> ans =
            new ArrayList<>();

    int i = 0;

    while(i < str.length())
    {
        int j = i;

        while(str.charAt(j) != '#')
        {
            j++;
        }

        int len =
                Integer.parseInt(
                        str.substring(i, j)
                );

        String word =
                str.substring(
                        j + 1,
                        j + len + 1
                );

        ans.add(word);

        i = j + len + 1;
    }

    return ans;
}
```

# Mistakes / Learnings

## Mistake 1

Initially tried:

```text
ab#xyz#
```

using delimiters.

Fails because:

```text
Strings may contain '#'.
```

---

## Mistake 2

Wrote:

```java
strs.length()
```

for List.

Correct:

```java
strs.size()
```

---

## Mistake 3

Wrote:

```java
strs.get(i).size()
```

for String.

Correct:

```java
strs.get(i).length()
```

---

## Mistake 4

Wrote:

```java
sb.append(#)
```

Correct:

```java
sb.append('#')
```

or

```java
sb.append("#")
```

---

## Mistake 5

Forgot to use:

```java
sb.append(length)
```

before appending the string.

Without storing length, decoding is impossible.

---

## Mistake 6

Initially returned:

```java
return s;
```

instead of:

```java
return sb.toString();
```
